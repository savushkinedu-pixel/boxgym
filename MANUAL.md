# BoxGym — Инструкция пользователя

## Атлет

### Первый вход
Напиши боту `/start` → выбери роль **Атлет** → появится меню кнопок.

### Меню кнопок
После входа внизу экрана появляется постоянная клавиатура:

```
┌─────────────────┬─────────────────┐
│  📅 Расписание  │  📋 Мои записи  │
├─────────────────┼─────────────────┤
│  💳 Баланс      │  📊 История     │
└─────────────────┴─────────────────┘
```

| Кнопка | Что показывает |
|--------|----------------|
| 📅 Расписание | Тренировки на текущую неделю с кнопкой [Записаться] |
| 📋 Мои записи | Предстоящие записи с кнопкой [Отменить] |
| 💳 Баланс | Тип абонемента, остаток визитов, срок действия |
| 📊 История | Последние 10 транзакций (списания, начисления) |

### Запись на тренировку
1. Нажми **📅 Расписание**
2. Найди нужную тренировку → нажми **[Записаться]**
3. Если мест нет — кнопка будет серой **[❌ Мест нет]**

### Массовая запись — `/subscribe`
Записаться сразу на все повторяющиеся тренировки на 4 недели вперёд:
1. Напиши `/subscribe`
2. Выбери дни недели (Пн, Вт, Ср…) → нажми **Готово →**
3. Введи время в формате `09:30`
4. Бот запишет тебя на все подходящие тренировки и сообщит сколько записей создано

### Списание визитов
- Визит списывается **автоматически** — через 2 часа после начала тренировки
- Ты ничего не должен делать: просто ходи на тренировки
- Когда визиты заканчиваются — бот пришлёт уведомление
- Посмотреть остаток: кнопка **💳 Баланс**

### Заморозка абонемента — `/freeze`
1. Напиши `/freeze`
2. Бот отправит заявку администратору
3. Администратор свяжется с тобой для подтверждения

### Моя статистика — `/mystats`
Показывает личную статистику:
- Визитов за текущий месяц
- Всего визитов за всё время
- Серия дней подряд 🔥

### Подтверждение оплаты
1. Оплати наличными или переводом
2. Отправь фото чека прямо в бот
3. Тренер получит уведомление с фото и кнопками **[✅ Подтвердить]** / **[❌ Отклонить]**
4. После подтверждения — администратор начислит абонемент в веб-панели

## Тренер

### Первый вход
Напиши боту `/start` → выбери роль **Тренер**.

### Мои группы — `/mygroup`
Показывает список записанных атлетов по периодам:
- **Сегодня** — тренировки текущего дня + кто записан
- **Завтра** — тренировки следующего дня
- **Эта неделя** — остальные тренировки недели

### Пригласить атлета — `/invite`
Генерирует одноразовую реферальную ссылку для нового атлета:
1. Напиши `/invite`
2. Бот пришлёт ссылку вида `t.me/boxgymbot?start=invite_XXXXXXXX`
3. Отправь ссылку атлету — при переходе он сразу зарегистрируется и будет привязан к тебе

### Моя статистика — `/mystats`
Показывает статистику по твоим тренировкам:
- Тренировок проведено за текущий месяц
- Средняя заполняемость групп (%)

### Отметка посещений — `/attendees` (опционально)
Позволяет вручную отметить кто пришёл:
1. Напиши `/attendees`
2. Бот покажет список записанных на ближайшую тренировку
3. Нажми **[✅ Пришёл]** или **[❌ Не пришёл]** для каждого атлета

> **Важно:** ручная отметка необязательна. Если тренер не отметил — система автоматически спишет визиты всем записанным атлетам через 2 часа после начала тренировки.

### Логика автосписания
- Каждые 5 минут система проверяет тренировки, у которых `начало + 2 часа ≤ текущее время`
- Все букинги со статусом `booked` автоматически переводятся в `attended`
- Визит списывается с абонемента атлета, создаётся запись в истории транзакций
- Букинги со статусом `cancelled` не трогаются — отмена защищает от списания
- Атлет получает уведомление в Telegram если визиты закончились

## Администратор

Веб-панель: **https://boxgym.edsaw.cc**

### Дашборд — `/dashboard`
- Общая статистика: число атлетов, активные абонементы, визиты сегодня/за месяц
- График посещаемости (неделя / месяц)
- Топ тренировок по заполняемости
- Атлеты, не посещавшие зал более 14 дней

### Расписание — `/schedule`
- Создание тренировок (тип, тренер, дата/время, вместимость)
- Просмотр всех тренировок на неделю

### Атлеты — `/athletes`
- Таблица всех атлетов с балансом абонементов
- Фильтр должников (нет активного абонемента)
- Кнопка **[+ Начислить]** — создать или пополнить абонемент
- Экспорт истории транзакций атлета в CSV

### Заморозка абонементов
- При запросе `/freeze` от атлета — администратор получает уведомление в Telegram
- Заморозить/разморозить абонемент можно через API: `PATCH /memberships/:id/freeze`
